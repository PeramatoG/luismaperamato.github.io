Plugins para GrandMA3

Palgen – Color Palette Generator para GrandMA3

GrandMA3

Plugin para GrandMA3 que genera paletas de color para rangos de grupos usando diferentes modos de armonía de color (análoga, complementaria, tríada, etc.). Incluye una ventana propia con faders y campos numéricos para configurar HUE, saturación, aleatoriedad y rango de grupos.

Versión actual: 1.0.0 · Proyecto abierto en GitHub.

Ver plugin en GitHub Ir al release v1.0.0 (Palgen.zip)

Label Colors – HTML colors para GrandMA3

GrandMA3

Plugin para GrandMA3 que rellena y renombra presets de color usando nombres estándar de colores HTML. Útil para crear una paleta de color coherente para los botones de cues y presets.
